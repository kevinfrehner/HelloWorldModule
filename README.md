# Hello World

This project uses the programming language, Python, in order to execute a program that displays the message "Hello World!" to the user. This project is typically used as the foundation for learning any new language.

## Instructions for Build and Use

Steps to build and/or run the software:

1. Open VS Code or other IDE.
2. Open the selected file.
3. Click 'Run Python File' to display the message to the console.

Instructions for using the software:

1. Download any version of python.
2. Install your choice of IDE.
3. Download this folder and open in the selected IDE.

## Development Environment

To recreate the development environment, you need the following software and/or libraries with the specified versions:

* Python (3.11)
* VS Code (1.111.0)
